Dotfiles 
----

My dot files for:
- bash
- zsh
- oh my zsh!
- tmux
- tmuxinator
- vim
- htop
- inputrc
- matplotlib
- jupyter (former known as ipython notebook)

## Requirements

Only GNU `stow` is required to manage this dotfiles repo.

## Usage

    git clone https://github.com/pablocobelli/Dotfiles.git
    cd Dotfiles
    stow bash
    stow zsh
    ...
    stow vim

Stow away at will!
